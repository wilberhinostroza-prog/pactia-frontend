# Pactia

**Préstamos de confianza entre personas. Registro inmutable en Algorand.**

## Colores
- Verde olivo `#556B2F` → Confianza, crecimiento
- Azul marino `#2C3E50` → Seguridad, seriedad

## Stack
- Frontend: React Native + Expo
- Backend: Node.js + Express
- Blockchain: Algorand
- Pasarela: Koywe (simulado)

## Comandos rápidos
```bash
# Frontend
cd frontend/Pactia && npx expo start

# Backend
cd backend && npm run dev
